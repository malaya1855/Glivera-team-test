## Запуск проєкту

1. Склонуйте репозиторій:

```bash
git clone https://github.com/malaya1855/test-Ignat-React.git
```
2. Перейдіть в папку ```Compiled-HTML``` - ```index.html``` 
3. Запустіть Go Live Server
